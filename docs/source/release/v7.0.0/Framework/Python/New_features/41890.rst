- :py:obj:`MDHistoWorkspace <mantid.dataobjects.MDHistoWorkspace>` now provides ``setNumEventsArray`` to overwrite the number-of-events array from a numpy array, mirroring the existing ``setSignalArray`` and ``setErrorSquaredArray`` methods. Manual updates through these array setters clear any link to the original ``MDEventWorkspace`` so downstream rebinning or slicing cannot bypass the manually-set values.
